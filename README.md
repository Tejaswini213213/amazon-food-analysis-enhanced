# Amazon Fine Food Reviews - Sentiment Analysis (Enhanced Project)

## 📌 Overview
This project performs sentiment analysis on the Amazon Fine Food Reviews dataset using Natural Language Processing (NLP) and Machine Learning techniques.

It is an enhanced version of an existing open-source analysis, improved for better structure, preprocessing, and insights.

---

## 🎯 Objective
- Analyze customer reviews and classify sentiment (positive/negative)
- Perform exploratory data analysis on review patterns
- Build machine learning models for sentiment prediction

---

## 📊 Dataset
- Amazon Fine Food Reviews (Kaggle)
- Contains customer reviews, ratings, and metadata
- Focus: text-based sentiment classification

---

## 🛠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- NLTK
- Matplotlib, Seaborn
- Jupyter Notebook

---

## 🚀 My Contributions
- Improved text preprocessing (cleaning, stopwords removal, tokenization)
- Enhanced exploratory data analysis and visualizations
- Structured notebook into clear ML pipeline stages
- Improved readability and documentation of workflow

---

## 📈 Models Used
- Logistic Regression
- Naive Bayes
- Support Vector Machine (optional depending on notebook)

---

## 📌 Key Insights
- Most reviews are positive, indicating high customer satisfaction
- Negative reviews highlight product quality and delivery issues
- Text preprocessing significantly improves model performance

---

## ⚠️ Note
This project is based on an open-source dataset analysis and enhanced with improvements for learning and portfolio purposes.
